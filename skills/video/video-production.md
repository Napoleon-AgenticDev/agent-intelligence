# Video Production Agent Skill

## Purpose
Enable AI agents to produce videos from concept to completion, including script, generation, editing, and publishing.

## Triggers
- User asks to create a video
- Video script needed
- Video editing requested
- YouTube/TikTok publishing needed

## Workflow

### Phase 1: Pre-Production
1. Define video concept and goal
2. Identify target audience
3. Choose video style (promotional, educational, narrative)
4. Plan duration and structure
5. Write detailed script

### Phase 2: Production
1. Generate visual assets (images/video clips)
2. Generate narration/voice-over
3. Generate background music
4. Ensure visual consistency

### Phase 3: Post-Production
1. Edit and compose final video
2. Add subtitles and captions
3. Mix audio (voice, music, SFX)
4. Export in required formats

### Phase 4: Distribution
1. Create thumbnails
2. Write descriptions and tags
3. Publish to platforms
4. Track performance

## Common Tasks

### Script Writing
- Story structure
- Scene breakdown
- Narration writing
- Caption creation

### Video Generation
- Image-to-video
- Text-to-video
- Keyframe generation
- Style consistency

### Audio Production
- Voice-over generation
- Music generation
- Sound effects
- Audio mixing

### Editing
- Scene composition
- Timing and pacing
- Transitions
- Color correction

## Tools
- Video generation (Hailuo, Veo, Runway, Pika)
- TTS services (ElevenLabs, MiniMax, Coqui)
- Music generation (Suno, MiniMax)
- Editing (ffmpeg, Premiere, DaVinci)

## Rules
- Always define the goal first
- Check for brand consistency
- Optimize for platform requirements
- Include captions for accessibility
- Track performance metrics

## Examples

### Create Explainer Video
```
User: Create a 60-second explainer for our SaaS product
Agent: 1. Write script (hook, problem, solution, CTA)
       2. Generate visuals matching brand
       3. Generate voice-over
       4. Add background music
       5. Edit and export
       6. Create thumbnail
```

### Short-Form Content
```
User: Create a TikTok about our new feature
Agent: 1. Write hook (first 3 seconds)
       2. Generate visuals
       3. Add trending music
       4. Add captions
       5. Export 9:16 format
```
