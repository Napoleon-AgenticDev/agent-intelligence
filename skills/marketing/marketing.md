# Marketing Agent Skill

## Purpose
Enable AI agents to execute marketing tasks across content, SEO, paid ads, and analytics with structured workflows.

## Triggers
- User asks to create marketing content
- SEO audit or optimization requested
- Ad campaign setup needed
- Analytics review requested
- Email sequence creation

## Workflow

### Phase 1: Strategy
1. Define target audience and goals
2. Research competitor strategies
3. Identify key messages and positioning
4. Choose channels and tactics

### Phase 2: Creation
1. Write compelling copy
2. Design visual assets if needed
3. Set up tracking and measurement
4. Configure automation workflows

### Phase 3: Execution
1. Publish to selected channels
2. Set up campaigns and targeting
3. Configure retargeting
4. Schedule follow-up content

### Phase 4: Measurement
1. Track key metrics
2. Analyze performance data
3. Identify optimization opportunities
4. Document learnings

## Common Tasks

### Content Marketing
- Blog post writing
- Social media content
- Email newsletters
- Case studies

### SEO
- Keyword research
- On-page optimization
- Technical audits
- Link building strategies

### Paid Ads
- Google Ads campaigns
- Meta Ads setup
- LinkedIn Ads
- Campaign optimization

### Analytics
- GA4 configuration
- Event tracking setup
- Conversion tracking
- Performance reporting

## Tools
- Analytics platforms (GA4, Looker Studio)
- Advertising platforms (Google, Meta, LinkedIn)
- SEO tools (Ahrefs, SEMrush)
- Email platforms (Klaviyo, Mailchimp)
- Social media tools (Buffer, Hootsuite)

## Rules
- Always define success metrics upfront
- Research before creating
- Follow brand guidelines
- Test and iterate
- Document everything

## Examples

### SEO Audit
```
User: Run an SEO audit on our landing page
Agent: 1. Analyze page content
       2. Check meta tags and headers
       3. Review internal linking
       4. Check page speed
       5. Analyze mobile usability
       6. Document findings and recommendations
```

### Ad Campaign
```
User: Set up a Google Ads campaign for our SaaS product
Agent: 1. Research keywords
       2. Define target audience
       3. Write ad copy
       4. Set up conversion tracking
       5. Configure bidding strategy
       6. Launch campaign
```
